# MCF_EARTH_PERTURB

Целью данной лабораторной работы является вычисление возмущающего ускорения, обусловленного нецентральностью гравитационного поля Земли, которое определяется на основе разложения геопотенциала по сферическим функциям. Также в работе выполняется построение и анализ графиков, позволяющих лучше понять влияние данных возмущений.

## ⚙️ Что реализовано

1) Решение уравнения Кеплера

    * Итерационный метод для нахождения эксцентрической аномалии
    * Вычисление истинной аномалии и радиус-вектора КА

2) Преобразование систем координат

    * Переход из абсолютной геоцентрической экваториальной (АГЭСК) в геоцентрическую (ГСК)
    * Учёт вращения Земли через угол $S(t)$
    * Переход к геоцентрическим сферическим координатам (ГССК): $(r, λ, φ)$

3) Рекуррентное вычисление полиномов Лежандра

    * Обычные полиномы $P_n(sin φ)$
    * Присоединённые функции $P_n^k(sin φ)$
    * Производные $\frac{∂P_n^k}{∂(sin φ)}$

4) Вычисление возмущающего ускорения

    * Радиальная компонента ($j_r$)
    * Широтная компонента ($j_φ$)
    * Долготная компонента ($j_λ$)
    * Орбитальные компоненты ($S$, $T$, $W$)

5) Валидация данных

    * Дескрипторы и properties для проверки корректности входных параметров
    * Проверка диапазонов орбитальных элементов

<a id="-основные-задачи-проекта"></a>
## 📈 Основные задачи проекта

1) Моделирование гравитационного поля

    * Реализация разложения потенциала по сферическим функциям
    * Учёт зональных гармоник ($J_n$)
    * Учёт тессеральных и секториальных гармоник ($C_{nk}$, $S_{nk}$)

2) Анализ сходимости

    * Исследование зависимости точности от числа учитываемых гармоник ($n_{max}$)

3) Зависимость от высоты

    * Построение графиков компонент ускорения от высоты КА
    * Определение точек экстремумов и других особенностей полученных графиков

4) Сравнение с центральным полем

    * Вычисление отношения возмущающего ускорения к центральному гравитационному
    * Оценка практической значимости возмущений ($0.01–0.1$%)

5) Влияние вращения Земли

    * Анализ изменения ускорения при возвращении в ту же точку орбиты

## ⚙️ Установка

1. **Клонируйте репозиторий**:
```bash

```

```bash

```
3. **Запустите программы**:

